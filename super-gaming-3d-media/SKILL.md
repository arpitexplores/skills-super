# Super Gaming / 3D / Media

## Overview
Build game systems, 3D experiences, and media pipelines with clear constraints.


## User Intent Examples
- "Need help with Game Development for my product/site."
- "Create a plan for Three.js & 3D Web."
- "Audit or improve Video/Media Tooling."

## Workflow
1. Define platform, performance targets, and content type.
2. Select engine or rendering stack.
3. Design assets, interaction, and performance budget.
4. Implement animation/audio/video workflows.
5. Test on target devices and optimise.
6. Document asset pipeline and build steps.

## Minimal Intake Questions
- Primary goal or outcome
- Scope (pages, systems, teams, or timeframe)
- Constraints (tools, budget, timeline)

## Output Format
- Platform and stack choice
- Asset and interaction plan
- Performance budget
- Media pipeline steps
- Testing checklist
