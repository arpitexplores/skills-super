# Super Office / Docs / Presentation

## Overview
Create and manage documents, spreadsheets, and presentations with repeatable workflows.


## User Intent Examples
- "Need help with Documentation Systems for my product/site."
- "Create a plan for Office Productivity."
- "Not sure where to start, need a quick assessment."

## Workflow
1. Identify the document type and intended audience.
2. Gather source material and required formats.
3. Draft or transform content with correct formatting.
4. Validate structure, readability, and compliance.
5. Export and document update process.

## Minimal Intake Questions
- Primary goal or outcome
- Scope (pages, systems, teams, or timeframe)
- Constraints (tools, budget, timeline)

## Output Format
- Document or presentation outline
- Formatting and export checklist
- Change log or update notes
