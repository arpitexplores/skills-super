## Why SUPER Skills For AI Vibe Coding

Super SEO Growth turns AI-assisted coding sessions into an SEO growth workflow. Use it when you want an agent to improve content quality, build topical authority, plan programmatic SEO, optimise for AI Overviews and LLM citations, and create measurable organic growth plans without guessing.

These skills are designed for **AI vibe coding**: fast, agent-assisted building where the AI needs strong domain context, practical workflows, guardrails, and implementation-ready outputs.

**SEO and discovery keywords:** AI SEO, GEO, generative engine optimisation, LLM visibility, AI Overviews, AI search visibility, programmatic SEO, content optimisation, topical authority, AI vibe coding, agent skills.

## Install

Copy this folder into your agent's skills directory, then restart or reload the agent.

```bash
cp -R super-seo-growth ~/.your-agent/skills/
```

Use it by name:

```text
Use $super-seo-growth to help with this request.
```
