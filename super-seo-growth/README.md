# Super SEO Growth

Grow organic visibility with content quality, programmatic SEO, and AI/GEO citation readiness.

## Install

Copy this folder into your agent's skills directory, then restart or reload the agent.

```bash
cp -R super-seo-growth ~/.your-agent/skills/
```

Use it by name:

```text
Use $super-seo-growth to help with this request.
```

## Best For

- SEO growth planning
- AI/GEO visibility
- content refreshes
- programmatic SEO expansion

## Outputs

- Prioritised growth opportunities
- AI search readiness plan
- content and programme roadmap
- measurement cadence

| Module | Purpose |
