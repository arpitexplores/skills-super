# Examples

Use these prompts to test or demonstrate `super-ai-ml-ops`.

- `Use $super-ai-ml-ops to design evals for this chatbot.`
- `Use $super-ai-ml-ops to reduce LLM cost and latency.`
- `Use $super-ai-ml-ops to create a production monitoring plan for this agent.`

## Evaluation Checklist

- The skill asks only for missing context.
- The output routes to the correct module.
- The answer is actionable and specific.
- Assumptions are stated clearly.
- Risks, validation steps, or next actions are included where relevant.
